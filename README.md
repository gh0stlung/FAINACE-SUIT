⚡ WHAT THIS IS

FINANCE SUITE is a local-first finance system with a modern UI and modular architecture.

Designed for clarity, speed, and control — without relying on servers.

- ✅ No login
- ✅ No tracking
- ✅ Works offline
- ✅ Data stays on your device

---

🧠 CORE MODULES

🏠 RENTBOOK (Property Manager)

- Tenant management system
- Monthly rent cycle with billing
- Status tracking → Settled / Pending / Due
- Clean UI for quick actions

---

👛 WALLET (Money System)

- Income + Expense tracking
- Monthly flow with carry-forward balance
- Calendar-based activity view
- Cash flow visualization (ring chart)

---

🎯 UX SYSTEM

- Compact Layout: Fits maximum data in one screen
- No Duplication: Same value not shown twice
- Visual Priority: Large numbers = key info
- Fast Input: Optimized for quick entry
